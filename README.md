## 🚀 Usage
(no executables to download, build it first.)

On linux, you should be able to launch the executable like:

```bash
cd Timer2
./Timer
```

On windows, there's NO support. Sorry. It may change in the future.

## 🔨 Building

For linux users, i left a "timer.pro" file, which does all the job for you. You'll need just qt6-dev and any compiler.
Something like this:

```bash
cd Timer2
qmake6 timer.pro
make
```


On windows, idk.
